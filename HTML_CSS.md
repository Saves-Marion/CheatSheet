## Code type HTML

<!DOCTYPE html>
<html>
    <head>
        <meta charset="utf-8" />
        <title>Titre_Ma_Page_Visible_Onglet</title>
    </head>

    <body>
    //son code ici
    </body>
</html>

| Balise         | Description      | 
|:--------------:|:----------------:| 
| `<html>`       | Principale       | 
| `<head>`       | En-tête, info gé | 
| `<body>`       | Corps page       | 
| `<meta    />`  |                  | 
| `<linkvvvv />` | Lien             | 
| `<script>`     | Code javascript  | 
| `<style>`      | Code CSS         |
